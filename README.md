# Internship Management Portal

A responsive Internship Management Portal built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**. This project was developed as part of the Full Stack Development Internship assignment.

---

## 🚀 Live Demo

**Live URL:**
https://anjali7549.github.io/internship-management-portal-tailwind/

**GitHub Repository:**
https://github.com/Anjali7549/internship-management-portal-tailwind

---

## 📌 Features

- Responsive Dashboard
- Sidebar Navigation
- Mobile Menu Toggle
- Student Registration Form
- Student Management Table
- Student Search
- Student Filter
- Form Validation
- Dark Mode Toggle
- Theme Persistence using Local Storage
- Dashboard Statistics
- Modal Popup
- Toast Notifications
- Local Storage Integration
- Student CRUD (Create & Delete)
- Dynamic Dashboard Chart

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- Vanilla JavaScript

1. Clone the repository.

3. Open `index.html` using Live Server or your browser.
4. Explore the different pages and features.
